Hacked Hogwarts

Introduction

This project was the first mandatory assignment on the Frontend Design elective. The assignment had a fairly long list of requirements. Here are some of the main ones:

  • The user must be able to filter, sort and search for a student
  • The user must be able to expel students and asign students to different roles.
  • The system can be 'hacked' (double clicking on the h1), affecting blood-status algorithm, injected me in the student list and other fun stuff.

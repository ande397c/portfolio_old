Falling Food

Introduction

The game Falling Foood, was made in the animation theme on 1.sem. The project included the making of a game using HTML, CSS and Javascript and designing the screens and graphical elements using Adobe's illustrator. This was my first introduction to Javascript. The projected also introduced me to CSS animations. Here are some of the main points of the project:

  • Coding a single game level, with several in game features like: timer, lives and points.
  • Implementing sound effects and music.
  • Designing all in game graphical elements.

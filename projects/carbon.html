Carbon Tracker

Introduction

This project was conducted as a group project , and also in collaboration with Advice A/S. Advice wanted a prototype for an online calculation tool for web CO2 emission. The tool was primarily aimed at managers and other decision makers to help create awareness and facilitate dialogue about the topic. Here are some of the requirements:

  • The user must be able to insert an URL so they can test if the website is considered green or not.
  • The user must be able to select their industry so, potentially, they can make benchmark with other industries.
  • The user must be able to make parameter changes so they can see how that effects their score.

SneakiDong

Introduction

This project was as conducted as a exam project at the end of 2. sem. The overall project was to find a client and redesign the webpage. My group and I settled on a website called SneakiDong, which specialized in selling accessories and products for sneakers. Here are some of the main points of the project:

  • Redesign the webpage using Wordpress and a custom childtheme. This included implementing filtering functions for the products and using Wordpress as database for all the product data.
  • Produce video content for the client.
  • Produce a related documentation rapport which contained primary and secondary research, a business analysis, and strategy for some and SEO.
